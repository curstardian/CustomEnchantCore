# 기능
- 날카로움 10, 찌르기 10, 모든 보호 인챈트 10
- 찌르기 추가 피해 지상에서도 적용
- 밸런스 패치

# 추가할것
- 고대도시, 바스티온에서 날카로움 3~6 인챈트 또는 보호 2~5가 낮은 확률로 나옴
- 난파선에서 아주 아주 낮은 확률로 찌르기 3~6이 나옴
- 도끼의 Sharpness 초과분에 추가 보너스를 없애기


# 문제점
-  String.join("\n", encoded) = GZIP or shorter encoding이 안되서 lore 많아지면 → PDC 길이 증가 아이템 복사 시 NBT 커짐 → 서버 렉 원인 가능
- setRepairCost(39) → XP 39 고정됨 사실상 모든 고레벨 인챈트가 너무 싸짐. 결과:경제 서버면 밸런스 붕괴 유저가 쉽게 10렙 찍음. 기본 마인크래프트에 레벨 비용 시스템 만큼 레벨을 내게 하고, 비용이 40을 넘으면 발생하는 "too expensive"이 나지 않게 만들어서 고쳐줘.
- LootGenerateEvent : 15%는 너무 높음. 0.05로
